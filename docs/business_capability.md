# Business Capability Definition

## 1. Overview
- system_name: 棚割管理システム (Planogram Management System)
- domain: リテール / 売場棚割管理
- description: 小売チェーンにおける棚割（プラノグラム）の標準化・個店展開・分析を一元管理するシステム。FMT（フォーマット）単位の標準棚割を基に、各店舗の什器サイズに応じた自動調整（カット・拡張）を行い、売場づくりの効率化と品質均一化を実現する。

| capability_id | capability_name | parent_capability_id | description |
|---------------|----------------|----------------------|-------------|

| CAP-002 | 棚ブロック管理 | - | 棚割の構成部品となるブロック（商品グループテンプレート）を管理する |

## 3. Capability Detail

### CAP-001
- name: マスタデータ管理
- description: 商品マスタ（JAN・寸法・売上ランク・8階層分類）、店舗マスタ（FMT・地域）、什器マスタ（タイプ・寸法・段数）、商品階層マスタを一元的に登録・更新・削除する
- input: 商品情報、店舗情報、什器情報、階層情報
- output: 各マスタの最新データ
- kpi: マスタ登録件数、データ整合率
- related_functions: [BF-001, BF-002, BF-003, BF-004, BF-005]

### CAP-002
- name: 棚ブロック管理
- description: 棚割の構成単位となるブロック（多段・平台）を作成し、ブロック内に商品を配置する。ブロックは標準棚割の構成部品として再利用される
- input: ブロック定義（名前・タイプ・寸法・段数）、商品配置情報
- output: 棚ブロックデータ
- kpi: ブロック作成数、ブロック再利用率
- related_functions: [BF-006]

### CAP-003
- name: 標準棚割管理
- description: FMT（MEGA/SuC/SMART/GO/FC）ごとの基準となる棚割をブロックの組み合わせで設計する。適用期間・什器タイプを指定し、個店展開の基盤となる
- input: FMT選択、基準店舗、什器タイプ、ブロック配置
- output: 標準棚割データ
- kpi: FMTごとの標準棚割数、標準棚割カバー率
- related_functions: [BF-007, BF-008]

### CAP-004
- name: 個店棚割管理
- description: 標準棚割を基に、各店舗の什器幅に応じてルールA（カット）/ルールB（拡張）を自動適用し、個店棚割を一括生成する。手動調整・同期機能も提供する
- input: 標準棚割、対象店舗一覧、店舗什器寸法
- output: 個店棚割データ（ステータス・警告付き）
- kpi: 自動生成成功率、手動調整率、同期率
- related_functions: [BF-009, BF-010, BF-011, BF-012]

### CAP-005
- name: 売場レイアウト管理
- description: 店舗フロア上に什器を2Dグリッドで配置する。ドラッグ&ドロップ、回転、衝突検知（AABB）、ゾーン・ラベル付けに対応する
- input: 店舗選択、什器選択、配置座標
- output: 店舗什器配置データ
- kpi: 什器配置完了店舗数
- related_functions: [BF-013]

### CAP-006
- name: 分析・可視化
- description: 売上・粗利・客数・客単価等のKPIをダッシュボードで表示し、ヒートマップ（JAN/階層/ブロック/棚割レベル）で棚割パフォーマンスを可視化する
- input: 分析スコープ（全店/個店）、メトリクス選択、集計レベル
- output: KPIカード、カテゴリ別チャート、ヒートマップ
- kpi: -
- related_functions: [BF-014, BF-015]

### CAP-007
- name: 帳票出力
- description: 店舗向けの棚割指示書を生成する。フロアレイアウト・什器別棚割・ブロック詳細・商品一覧を含む印刷用ドキュメントを出力する
- input: 対象店舗、注記・注意事項
- output: 棚割指示書（PDF印刷用）
- kpi: 指示書発行数
- related_functions: [BF-016]

### CAP-008
- name: データ入出力
- description: 商品マスタのExcel/CSVインポート（バリデーション・プレビュー・重複処理付き）、CSVエクスポート、階層マスタCSVインポートに対応する
- input: Excel/CSVファイル
- output: インポート結果、エクスポートファイル
- kpi: インポート成功率
- related_functions: [BF-017, BF-018]
