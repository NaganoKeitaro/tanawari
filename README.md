# 棚割管理システム (Planogram System)

React + TypeScript + Supabase で構築された、FMT標準棚割から個店棚割への自動展開機能を持つ棚割管理システムです。

## 特徴

*   **FMT標準棚割管理**: マスタとなる標準棚割をドラッグ＆ドロップで作成（什器タイプ別対応）
*   **個店展開の自動化**: 店舗ごとの棚サイズに合わせて、商品フェイス数を自動調整（カット/拡張）
*   **レスポンシブな棚ビジュアル**: 商品サイズ（mm/尺）に基づいた正確な視覚化・カテゴリ色分け
*   **商品ツールチップ**: マウスホバーで商品名・JAN・フェイス数・品種を即時表示
*   **分析ダッシュボード**: 売上・粗利・客数のKPI表示、ヒートマップ分析
*   **棚割指示書**: PDF出力対応の指示書生成（注釈・コメント付き）
*   **デュアルバックエンド**: Supabase（PostgreSQL）本番環境 + LocalStorage開発環境

## Getting Started (起動方法)

### 前提条件

*   Node.js (v18以降推奨)
*   npm

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてください。

### 環境変数（任意）

Supabaseを使用する場合、`.env` ファイルに以下を設定:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

未設定の場合は自動的にLocalStorageにフォールバックします。

### ビルド

```bash
npm run build
```

## 使い方 (Quick Start)

1.  **店舗マスタの生成**:
    *   ホーム画面 (`/`) の「店舗マスタ生成」ボタンでサンプル店舗データを生成します。

2.  **マスタデータの登録**:
    *   商品マスタ: Excel/CSVインポートまたは手動登録（JAN・商品名・サイズ・売上ランク）
    *   什器マスタ: 什器タイプ（多段棚・平台冷蔵・平台冷凍等）ごとにサイズ・段数を登録
    *   店舗棚尺マスタ: 各店舗に什器を2Dレイアウトで配置

3.  **棚ブロックの作成**:
    *   「棚ブロック管理」で商品をドラッグして棚ブロック（テンプレート）を作成します。

4.  **標準棚割の作成**:
    *   「FMT標準棚割」で棚ブロックを配置して標準パターンを作成します。

5.  **個店棚割の自動生成**:
    *   「個店棚割管理」で一括自動生成を実行します。
    *   店舗の棚幅に応じてカット/拡張ルールが自動適用されます。

6.  **棚割指示書の出力**:
    *   「棚割指示書」で店舗を選択してPDF出力します。

## 自動調整ルール

*   **ルールA (Cut)**: 店舗の棚が標準より**狭い**場合、売上ランク下位の商品からフェイス数を削減または削除
*   **ルールB (Expand)**: 店舗の棚が標準より**広い**場合、売上ランク上位の商品のフェイス数を増やしてスペースを埋める
*   **ルールC (Sync)**: 「同期」ボタンにより、標準棚割の最新構成を個店棚割に再適用
